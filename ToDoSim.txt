# ModHPS
Model for Heat Pump Systems (ModHPS)

0) optional: Change input-data in XML (folder 'ModHPS/import') 
1) Open folder 'ModHPS/src' in terminal
2) Kompilieren:
type 'g++ -std=c++11 -o ../ModHPS *.cpp' //'g++ -o ../a.out heatpump.cpp hplink.cpp hps.cpp parser.cpp hpcore.cpp building.cpp UseGNU.cpp PlottingSettings.cpp'
2A) 
for co-simulation with an included python script: type 'g++ -I/usr/include/python2.7 -includePython.h -o ../ModHPS.out heatpump.cpp hplink.cpp hps.cpp parser.cpp hpcore.cpp building.cpp UseGNU.cpp PlottingSettings.cpp -lpython2.7 -L/usr/lib/python2.7'
3) Open folder 'run' in terminal
4) type '../ModHPS'
5) results in folder 'ModHPS/run/hpm'
